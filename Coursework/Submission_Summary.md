# Submission Summary for CoU-MIS Project

## Project Title
Church of Uganda Management Information System (CoU-MIS)

## Course
CIS 7205 Advanced Database Systems

## Deliverables
- **Project Report**: `CoU-MIS_Project_Report.docx`
- **Presentation**: `CoU-MIS_Presentation.pptx`
- **Database Schema**: `Coursework/sql/schema.sql`
- **Sample Queries**: `Coursework/sql/sample_queries.sql`
- **Prototype Application**: `Coursework/app/`
- **Supporting Documentation**: markdown files in `Coursework/`

## Task Coverage
- Requirements analysis and stakeholder documentation
- ER design with entities, attributes, and relationships
- Relational schema design and 3NF normalization
- PostgreSQL implementation with tables, keys, and constraints
- Advanced queries using joins, subqueries, aggregates, and views
- UI prototype with CRUD operations for members, sacrament records, and transactions
- Security and integrity design through constraints and role-based structure

## Notes
- The prototype currently runs in-memory for demonstration.
- The SQL schema is ready for PostgreSQL deployment.
- The report includes a final checklist confirming completed coursework tasks.
